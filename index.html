MathToolkit Home Page
Patrick Ząbkiewicz, Oct 2014

   --- Introduction ---

MathToolkit is collection of structures, classes, wrappers, allocators, collectors and in-life examples which makes it basic tool for math scientist and engineers that are in deep need for open source and completly free tool that is designed and implemented with thought of scientific research.

Project is in early stage of development now. I need contributors who will make this dream library come true.

Project is licensed under LGPL 3.0 license.

   --- Download ---

MathToolkit source code can be downloaded from GitHub.com:
https://github.com/ComputerScienceImageAnalysisCircle/MathToolkit

Compilation process is done using cmake and make. For compilation on Windows you can use cygwin or cmake and mingw. The binaries for iOS and Android are provided via cross-compilation.
